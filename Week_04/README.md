#### 几种字符串分析算法：
* 字典树
	* 用于大量高重复字符串的存储与分析
* KMP
	* 在长字符串里找模式
* Wildcard
	* 带通配符的字符串模式
* 正则
	* 字符串通用模式匹配（字符串模式匹配的终极版本）
* 状态机（比正则表达式更强大，理论上，有限状态机和正则表达式是完全等价的两种东西，但是有限状态机不同的是，你还可以往里面嵌入代码，对字符串做额外的处理。有限状态机写起来成本比较高）
	* 通用的字符串分析
* LL LR
	* 字符串多层级结构分析
